# homeworks

Личный репозиторий с домашними заданиями по Python — от первых скриптов
с переменными до модульного CLI-приложения с типизацией. Каждая папка —
отдельное самостоятельное задание, без общего пакета и точки сборки.

## Стек

- **Python 3.14**, только стандартная библиотека — без внешних зависимостей
- Типизация через `typing` (`TypedDict`, `Literal`, дженерики) под строгий
  режим Pylance
- `match`/`case`, циклы, `sys.argv`-CLI, работа с `list`/`dict`/`set`

## Требования

- Установленный Python 3.14+ (можно проверить: `python --version`)
- Ничего больше не требуется — виртуальное окружение и файл зависимостей
  не используются

## Структура

Задания разделены по уровням сложности курса: `basic/`, `intermediate/`,
`advanced/`.

| Папка                     | Тема                                                       |
| ------------------------- | ----------------------------------------------------------- |
| `basic/1_vars/`           | Переменные, ввод/вывод                                     |
| `basic/2_operations/`     | Числовые операции, списки и кортежи                        |
| `basic/3_menu/`           | Ветвление, `match`/`case`                                  |
| `basic/4_expenses/`       | Интерактивное меню на циклах                                |
| `basic/5_library/`        | CLI-скрипт на `sys.argv` (`filter`/`sort` по каталогу книг) |
| `basic/6_crm/`            | Модульное приложение: домена (`orders.py`), хранилище, CLI |
| `intermediate/1_oop/`     | ООП (заготовка, `basic_oop.py`)                             |
| `advanced/`               | Пока пусто                                                  |

Подробное описание каждой папки и принятых соглашений — в [CLAUDE.md](./CLAUDE.md).

## Запуск

Каждый файл — самостоятельный скрипт:

```bash
python basic/1_vars/hello.py
python basic/4_expenses/menu.py
python basic/5_library/library.py filter "Идиот,Нос"
```

`6_crm` использует не полностью квалифицированные импорты, поэтому его
нужно запускать из собственной папки:

```bash
cd basic/6_crm
python __main__.py
```

## Статус

Работа продолжается — часть модулей (`6_crm/storage.py`, `6_crm/cli.py`,
`6_crm/utils/validators.py`) пока заглушки и будут дополняться по мере
прохождения тем.
